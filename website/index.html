Perguntas Frequentes

Quanto custa para estar antenado?
A Antena está atualmente em fase beta e é gratuita para uso. Futuramente, teremos planos pagos com recursos avançados, mas manteremos sempre uma versão gratuita com funcionalidades essenciais.
Com que frequência recebo os sinais?
Você pode configurar a frequência de recebimento dos sinais (diária, semanal ou mensal). Além disso, você pode solicitar informações específicas a qualquer momento usando os comandos disponíveis.
É possível desconectar minha antena quando quiser?
Sim, você pode cancelar sua inscrição a qualquer momento enviando o comando "/cancelar" no WhatsApp. Não há período mínimo de permanência ou taxas de cancelamento.
Quais comandos estão disponíveis no WhatsApp?
Os comandos principais são: "/tendencia [tópico]" para receber tendências, "/ferramenta [tópico]" para recomendações de ferramentas, "/case [tópico]" para casos de sucesso e "/ajuda" para ver todos os comandos disponíveis.
Como meus dados serão utilizados?
Utilizamos seus dados apenas para sintonizar os sinais mais relevantes para você e melhorar nosso serviço. Não compartilhamos suas informações com terceiros para fins de marketing. Para mais detalhes, consulte nossa Política de Privacidade.

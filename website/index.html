Perguntas Frequentes

Quanto custa para usar o Radar de Tendências?
O Radar de Tendências está atualmente em fase beta e é gratuito para uso. Futuramente, teremos planos pagos com recursos avançados, mas manteremos sempre uma versão gratuita com funcionalidades essenciais.
Com que frequência recebo as informações?
Você pode configurar a frequência de recebimento das informações (diária, semanal ou mensal). Além disso, você pode solicitar informações específicas a qualquer momento usando os comandos disponíveis.
É possível cancelar minha inscrição quando quiser?
Sim, você pode cancelar sua inscrição a qualquer momento enviando o comando "/cancelar" no WhatsApp. Não há período mínimo de permanência ou taxas de cancelamento.
Quais comandos estão disponíveis no WhatsApp?
Os comandos principais são: "/tendencia [tópico]" para receber tendências, "/ferramenta [tópico]" para recomendações de ferramentas, "/case [tópico]" para casos de sucesso e "/ajuda" para ver todos os comandos disponíveis.
Como meus dados serão utilizados?
Utilizamos seus dados apenas para personalizar sua experiência e melhorar nosso serviço. Não compartilhamos suas informações com terceiros para fins de marketing. Para mais detalhes, consulte nossa Política de Privacidade.
